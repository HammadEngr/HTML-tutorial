Links and Anchors

In HTML we have 4 tags for links

  1. <a> tag
  2. <area> tag
  3. <Link> tag, This is only used in head tag of document

Each of these tags have an attribute called "Hypertext Reference" which is written like "href"

Before proceeding towards links and anchors, you must understand URL first i-e Uniform Resource Locator
as the name indicates, it is a resource locator.
Resource can be

  • another website
  • a file like html file
  • section of the same document

based on type of resource, URL is classified in 3 types

  • Absolute URL
  • Relative URL
  • Internal URL
