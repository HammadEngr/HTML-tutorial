1. Attributes,       
HTML attributes are special words which are used inside the opening tag of
elements which can impart varius properties to tags and they can control
tags behaiviour. They either modify the default behaviour of ELEMENT or
can add some functionality to them.

2.Attributes can be classified as.
      1. Global Attributes. 
      2. Local Attributes. 
      3. Event Atrributes.

      1. Global Attributes are those which can be used in all HTML elements,
      <br />
      e-g: hidden,

      2. Local Attribbutes are those which are specific to particular HTML tags
      only, fo example we have an <img src="" alt="" />

      3. Event attributes are used to specify what actions browsers should take
      when the user interacts with an element. they are basically java script
      functions and since this course is about html only i will not go into
      details of this.
